# Large-Format On-Site Diagnostics

**Version:** 2026-06-18  
**Market:** South Africa / Southern Africa  
**Business:** Tiletoria customer and staff RAG knowledge base  

## Purpose
Help Tori identify risks before customers install large-format tiles such as 600x1200, 700x1400, 800x1600, 1200x1200 and larger.

## Retrieval triggers
- large format
- 600x1200
- 700x1400
- 800x1600
- 1200x1200
- big tiles
- lippage
- warped
- flat floor

## Large-format definition for Tori
Large-format tiles require more planning because small substrate issues become visible as lippage, hollow spots, cracked tiles or poor alignment.

## Ask before advising
- Tile size.
- Floor or wall.
- Indoor/outdoor/wet area.
- Substrate type and flatness.
- Room size and layout.
- Desired grout joint.
- Installer experience.
- Adhesive type.
- Levelling clips planned?

## Critical message
Levelling clips help align tile edges while adhesive cures. They do not fix an uneven floor, poor adhesive coverage or wrong preparation.

## Flatness warning
Large tiles need a flatter substrate than small tiles. Tori should advise checking substrate flatness before installation and using self-levelling or remedial preparation where needed.

## Layout planning
Recommend dry layout planning for:
- Centre lines.
- Doorways.
- Feature walls.
- Avoiding narrow cuts.
- Pattern direction.
- Mixed boxes/batches.

## Adhesive coverage
Large-format tiles require correct adhesive selection and good coverage. Back-buttering may be needed depending on product/application/manufacturer guidance.

## Lippage complaints
If a customer asks about lippage after installation, Tori must collect:
- Tile size.
- Area.
- Photos with straight edge if possible.
- Substrate prep.
- Adhesive used.
- Grout joint width.
- Clip/spacer used.
- Installer details.

## Stop rule
If large-format tiles are being installed on a visibly uneven floor, Tori must advise stopping and correcting the substrate before continuing.
