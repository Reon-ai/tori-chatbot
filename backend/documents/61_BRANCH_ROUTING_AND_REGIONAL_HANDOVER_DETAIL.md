## Branch routing principle
Route customers to the most relevant Tiletoria or Spec Lab location based on region, product need and customer type. Use official branch/contact files when available.

## Region routing
- Western Cape / Cape Town: Cape Town / Paarden Eiland route unless another WC branch is specified.
- Gauteng / Johannesburg / Pretoria: Northriding route; Menlyn references can route back to Northriding unless current official details say otherwise.
- KZN / Durban / Umhlanga / Cornubie: Cornubie/KZN route.
- Paarl / Winelands: Spec Lab Paarl route where relevant.

## Ask before routing if unclear
- Which province or suburb are you in?
- Is this for collection, delivery, showroom visit or project support?
- Are you looking for Tiletoria or Spec Lab assistance?

## Staff handover fields
- Customer name.
- Contact.
- Region/suburb.
- Preferred branch.
- Product category.
- Quote/technical/account/complaint/sample.
- Urgency.

## Avoid
Do not invent phone numbers, emails or trading hours. Use the approved branch contact Markdown file if present.

## Routing by customer type
- DIY/homeowner: nearest showroom/branch.
- Tiler/contractor: branch sales/trade counter.
- Developer/QS: project sales/key account.
- Architect/designer: showroom/specification contact.
- Complaint: branch manager/customer service.
- Account query: accounts.

---

## Universal Tori rule for this file
Tori must be helpful, practical and commercially safe. Tori may guide, calculate and explain, but must not invent live stock, live prices, delivery availability, technical certifications, warranties, legal outcomes or product suitability where site conditions are unknown. When the question depends on site inspection, current branch stock, quoted pricing, account status or formal compliance, Tori must capture the correct details and route to the correct Tiletoria team.
