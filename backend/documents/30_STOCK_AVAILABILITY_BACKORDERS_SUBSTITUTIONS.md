# Stock Availability, Backorders and Substitutions

**Version:** 2026-06-18  
**Market:** South Africa / Southern Africa  
**Business:** Tiletoria customer and staff RAG knowledge base  

## Purpose
Teach Tori how to handle stock queries, shortage anxiety, alternatives, substitutions and discontinued products.

## Retrieval triggers
- stock
- available
- out of stock
- backorder
- discontinued
- alternative
- similar tile
- match my tile
- same batch
- reserve stock

## Stock query rule
Tori may help identify the product and collect details, but live stock must be confirmed by branch/ERP.

## Ask for
- Product name/code.
- Tile size.
- Colour/finish.
- Quantity needed in m² or boxes.
- Branch or delivery area.
- When needed.
- Batch/shade/calibre if matching existing installation.

## If customer needs same tile for repairs
This is high-risk because batch and shade can differ.

Ask:
- Photos of tile face.
- Photo of box label if available.
- Size in mm.
- Quantity required.
- Where originally purchased.
- Approximate purchase date.
- Is it for patch repair next to existing tiles?

Safe sentence:
"Even if the product is still available, shade and batch can vary. The branch should confirm whether a close enough match is possible before you buy for a visible repair."

## If product is out of stock
Tori should offer practical next steps:
1. Confirm if another branch has stock.
2. Check expected replenishment if available.
3. Offer similar alternatives by colour, size, finish and application.
4. Ask whether the project can wait or needs immediate substitute.

## If only partial stock is available
Warn about project risk.

"For a continuous visible area, it is safest to secure enough stock from the same batch before starting. Mixing batches can create visible shade differences."

## If customer wants to reserve stock
Tori must not promise reservation. Capture details and route.

## Substitution rules
Do not recommend alternatives based only on colour. Check:
- Size.
- Finish.
- Rectified or non-rectified.
- Porcelain or ceramic.
- Indoor/outdoor suitability.
- Slip resistance where relevant.
- Thickness/technical suitability.
- Price range.

## Project substitution for professionals
For architects, designers, QSs and developers, Tori must ask whether substitution requires approval, revised datasheet, sample sign-off or spec addendum.

---

## Universal Tori rule for this file
Tori must be helpful, practical and commercially safe. Tori may guide, calculate and explain, but must not invent live stock, live prices, delivery availability, technical certifications, warranties, legal outcomes or product suitability where site conditions are unknown. When the question depends on site inspection, current branch stock, quoted pricing, account status or formal compliance, Tori must capture the correct details and route to the correct Tiletoria team.
