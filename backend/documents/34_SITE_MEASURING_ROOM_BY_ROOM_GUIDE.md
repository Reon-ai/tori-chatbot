# Site Measuring — Room by Room Guide

**Version:** 2026-06-18  
**Market:** South Africa / Southern Africa  
**Business:** Tiletoria customer and staff RAG knowledge base  

## Purpose
Give Tori a practical measurement workflow for customers who do not know their square metres.

## Retrieval triggers
- measure
- measurement
- how many tiles
- room size
- bathroom size
- kitchen size
- calculate m2
- square metres

## Tori's measurement goal
Turn vague project descriptions into usable m² quantities and quote-ready information.

## Basic floor formula
```text
Floor area = length × width
```

Use metres. Example:
```text
4.2 m × 3.6 m = 15.12 m²
```

Then add waste.

## Wall formula
```text
Wall area = wall width × wall height
```

For several walls, calculate each wall and add together.

## Door/window deductions
For simple customer estimates, Tori may say deductions are optional for rough budgeting. For final quantities, deduct large openings if accurate measurements are available.

## Bathroom measurement flow
Ask:
- Floor length and width.
- Wall tile height.
- Number of walls tiled.
- Shower walls included?
- Niche/feature wall/mosaic?
- Tile size.
- Waste allowance.

## Kitchen measurement flow
Ask:
- Floor length and width.
- Splashback width and height if applicable.
- Is there an island?
- Are tiles going under cupboards/appliances?

## Patio/balcony measurement flow
Ask:
- Length and width.
- Is it covered or exposed?
- Is there drainage/fall?
- Is there waterproofing below?
- Is it above a living area?
- Slip resistance required?

## Stairs measurement flow
Ask:
- Number of steps.
- Tread width and depth.
- Riser height if risers are tiled.
- Nosing/trim requirement.
- Slip risk.

## LVT measurement flow
Ask:
- Room length and width.
- Subfloor type and condition.
- Moisture concerns.
- Is skirting being removed?
- Are trims/profiles needed at doorways?

## Waste recommendation
Use practical ranges:
- Standard floor or wall: 10%.
- Small bathroom with many cuts: 12% to 15%.
- Diagonal/herringbone/patterned layout: 15% or more.
- Large-format with tight layout planning: 10% to 15%.

## Tori response template
"Please send me the length and width of each area in metres. I’ll calculate the m², add a sensible waste allowance and convert it into boxes if we know the m² per box."
