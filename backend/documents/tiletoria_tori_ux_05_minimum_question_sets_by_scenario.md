# Tori UX 05: Minimum Question Sets by Scenario

This file tells Tori the smallest useful question to ask in common scenarios.

Purpose: Improve UX by avoiding question overload.

Important rule: Ask one question first. Ask more only after the customer answers.

## Scenario: Bathroom Tile Selection

Summary: Identify floor, wall, shower or full bathroom first.

Minimum first question:
“Is this for the bathroom floor, shower floor, shower walls, or the whole bathroom?”

If answer is bathroom floor:
Ask next only if needed: “Is it a normal bathroom floor or inside a wet shower area?”

If answer is shower floor:
Escalate waterproofing, falls, drain and slip-aware guidance.

## Scenario: Outdoor Tile Selection

Summary: Exposure determines the next answer.

Minimum first question:
“Is the area covered, fully exposed to rain, or around a pool/ramp?”

If covered:
Discuss outdoor suitability if wind/rain can enter.

If exposed:
Discuss drainage, exterior adhesive, exterior grout and movement joints.

If balcony:
Escalate waterproofing and drainage.

If ramp:
Escalate slip-risk and project review.

Retrieve:
Product Suitability.
Outdoor Project Planning.
Cleaning Maintenance.

## Scenario: Glue-Down LVT

Summary: Subfloor is the first critical issue.

Minimum first question:
“Is the glue-down LVT going over screed, existing tiles, or another floor?”

If screed:
Ask about moisture and smoothness.

If existing tiles:
Discuss levelling grout lines and bonding.

If bathroom:
Escalate moisture and product suitability.

Retrieve:
Product Suitability.
Measurement Guide.
Cleaning Maintenance.

## Scenario: Tile Quantity

Summary: Area and tile size are needed first.

Minimum first question:
“What is the area in square metres and what tile size are you using?”

If area unknown:
Ask for room length and width.

If pattern:
Ask if straight, diagonal, herringbone or feature layout.

Retrieve:
Measurement Guide.

## Scenario: Adhesive Quantity

Summary: Area and tile size matter.

Minimum first question:
“What area are you tiling and what tile size is it?”

If outdoor or wet:
Ask if the adhesive is for wet/exterior use.

If large-format:
Mention substrate flatness and adhesive coverage.

Retrieve:
Measurement Guide.
Adhesive Q&A file.

## Scenario: Grout Quantity or Colour

Summary: Quantity needs tile size, area and joint width; colour needs design intent.

Minimum first question for quantity:
“What tile size, area and grout joint width are you using?”

Minimum first question for colour:
“Do you want the grout to blend with the tile or create contrast?”

Retrieve:
Measurement Guide.
Grout Q&A file.
Design file when available.

## Scenario: Toilet Replacement

Summary: Waste outlet matters first.

Minimum first question:
“Are you replacing an existing toilet, and does the waste outlet go through the floor or wall?”

If floor waste:
Ask set-out if needed.

If wall waste:
Ask whether P-trap/back-to-wall applies.

Retrieve:
Sanitaryware Q&A.
Add-On Shopping Lists.

## Scenario: Basin or Vanity

Summary: Basin installation type matters first.

Minimum first question:
“Is the basin going on a vanity, on a countertop, or directly onto the wall?”

If vanity:
Check trap clearance.

If countertop:
Check tall mixer and waste.

If wall-hung:
Escalate wall support.

Retrieve:
Sanitaryware Q&A.
Add-On Shopping Lists.

## Scenario: Tap or Mixer

Summary: Pressure and mounting type matter.

Minimum first question:
“Is this for a basin, bath, shower or kitchen?”

If shower:
Ask concealed or exposed.

If low-pressure concern:
Escalate plumber confirmation.

Retrieve:
Sanitaryware and Tapware Q&A.
Product Suitability.

## Scenario: Stock or Price

Summary: Branch and product code are needed.

Minimum first question:
“Which branch is closest to you, and what product name or code are you looking for?”

If no product code:
Ask for description or photo.

If no branch:
Ask customer location.

Retrieve:
Store Details.
Lead Capture Handover.

## Scenario: Complaint

Summary: Installed or uninstalled is the first fork.

Minimum first question:
“Is the product already installed, or is it still in the box?”

If installed:
Ask for invoice, photos and installation details.

If uninstalled:
Ask for box labels and packaging photos.

Retrieve:
Complaint Triage.

## Scenario: Installer Disagreement

Summary: The concern must be identified before advice.

Minimum first question:
“What exactly is the installer concerned about: product suitability, substrate, adhesive, waterproofing or plumbing?”

Rule:
Recommend pausing before proceeding.
