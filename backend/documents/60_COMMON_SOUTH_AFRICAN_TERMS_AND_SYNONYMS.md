## Purpose
Customers do not always use formal product names. This file helps Tori understand local terms and route correctly.

## Tile and flooring terms
- Tile cement = tile adhesive.
- Glue = adhesive.
- Grouting = grout.
- Spacers = tile spacers.
- Clips = levelling clips.
- Wedges = levelling wedges.
- Tile levellers = levelling clips/wedge system.
- Screed = cementitious floor layer/substrate.
- Self-leveller = self-levelling compound.
- Vinyl = usually LVT/SPC/vinyl flooring; Tiletoria focus is glue-down LVT.
- Click vinyl = not Tiletoria's glue-down LVT focus unless explicitly stocked.

## Application terms
- Stoep = patio/veranda/external covered area.
- Braai room = indoor/semi-outdoor entertainment area; check exposure.
- Scullery = utility/laundry/wet-service area.
- En-suite = bathroom attached to bedroom.
- BIC = built-in cupboards; affects measurement.
- Garage = vehicle area; check suitability.

## People terms
- Builder = contractor/foreman/building company.
- Tiler = installer/subcontractor/artisan.
- Plumber = sanitaryware/tapware/fittings/waste installation professional.
- QS = quantity surveyor.
- Managing agent = body corporate/HOA property manager.
- Assessor/loss adjuster = insurance claim evaluator.

## Common misspellings Tori should understand
- prochedure = procedure.
- publick = public.
- caqche = cache.
- screener/spacer confusion = likely tile spacer or levelling spacer.
- porcelin = porcelain.
- grout haze = residue after grouting.
- lippage = uneven tile edges.
- efflorence/effloresence = efflorescence.

## Product matching words
- Marble-look = marble effect.
- Screed-look = concrete-look/cement-look.
- Stone-look = natural stone effect.
- Wood-look = timber-look porcelain/vinyl.
- Matt = matte.
- Gloss = polished/shiny.

## Routing reminders
When customer uses vague terms like "outside", "wet", "slippery", "leaking", "cracked", "lifting" or "claim", Tori must route into the risk/technical flow, not only product sales.
